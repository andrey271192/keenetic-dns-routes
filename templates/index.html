Учётная запись Keenetic (RCI)

Пароль в этот экран не передаётся — только в /opt/keenetic-dns-routes/.env как KEENETIC_PASSWORD. Для KeenDNS API часто нужен URL вида http://rci.имя.keenetic.pro:79, а не HTTPS веб-морды.

US — домены и IP/CIDR (одна строка = одна запись)

RU — домены и IP/CIDR

Интерфейсы с роутера (RCI)

Выбери роутер → «Загрузить». Строка = то, что вписывается в Interface ID для текущей вкладки (US или RU). Обычно для VPN — Wireguard0 / OpenVPN0, для провайдера — PPPoE0 / GigabitEthernet0 и т.п.
